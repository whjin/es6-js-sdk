# 箭头函数 #

- `this`指向定义时所在对象
- 不可以作为构造函数（不能使用`new`）
- 没有`arguments`对象

# const & let #

- 块级作用域
- 使用`let`和`const`不再有变量提升，不允许重复声明
- 优先使用`const`

# 模块化 #

- 静态化，必须在顶部，不能使用条件语句，自动采用严格模式
- `treeshaking`和编译优化，以及`webpack3`中的作用域提升
- 外部可以拿到实时值，而非缓存值（是引用而不是拷贝）

<hr>

- 可以对`commonJS`模块重新赋值，对`ES6`模块重新赋值会编译报错
- 都可以对对象内部的值进行改变
- `commonJS`是对模块的拷贝
- `ES6`是对模块的引用
