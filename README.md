# es6-js-sdk
电商网站的账号体系

- 原生`ES6`开发，除兼容库外不引用任何第三方库
- 包括工程化、架构设计、数据`Mock`等常用前端开发模式与流程
- 账号体系`SDK`，同意控制核心功能，对下游开放样式定制

# ES6+ #

- 更高的开发效率 `Object.assign` `Array.from` ...
- 更少的出错可能 `let&const` `Symbol` ...
- 更人性化的语法 **箭头函数** `async/await` ...

# 开发工具 #

- 工程化工具`webpack`
- 语法编译工具`babel`
- 脚手架工具

## webpack配置 ##

- `webpack`用于支持**模块化开发**
- 配置`babel-loader`将`es6`转换为`es5`
- 压缩、合并等前端优化
